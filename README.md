# TXT Line Search DB

> **텍스트 파일을 줄(Line) 단위로 데이터베이스화하여 고속 검색하고 원문 위치를 탐색하는 프로그램**
> 
> 이 프로그램은 복잡한 논리 검색식(`and`, `or`, `&`, `()`)을 파싱하여 대용량 텍스트 파일 내의 특정 라인을 순식간에 찾아냅니다. 친구분을 위한 선물 또는 로컬 텍스트 분석 도구로 완벽하게 작동하며, 파이썬이 없는 환경에서도 단독 실행 파일(`.exe`) 하나로 실행 가능하도록 최적화되어 있습니다.

---

## 🚀 주요 기능

1. **줄(Line) 단위 전처리 및 SQLite DB화**
   - 텍스트 파일 업로드 시 인코딩 자동 감지(UTF-8-sig, CP949 등)
   - 불필요한 빈 줄을 제거하고 양쪽 공백을 정리하여 SQLite DB(`segments` 테이블)에 인덱스와 함께 자동 구축

2. **강력한 논리 검색식 파서 (Query Parser)**
   - 괄호 `()`, `and`, `or`, `&` 연산자가 포함된 복잡한 검색 문법 지원 (예: `(가나다and라마)or사아`)
   - 연산자 우선순위(`()` > `and` > `or`)를 완벽히 해석하여 SQL `LIKE` 조건문으로 이스케이프 안전 변환

3. **초고속 렌더링 성능 최적화 (가상 스크롤)**
   - **가상 스크롤(Virtual Scroll)**: 수만 줄짜리 원문 파일 진입 시 화면에 보이는 뷰포트 영역(30~50줄)만 실시간으로 DOM 노드에 그려주어 버퍼링과 브라우저 프리징을 원천 차단
   - **무한 스크롤(Infinite Scroll)**: 검색 결과가 수천 건 이상인 경우 100개씩 동적 페이징 렌더링을 지원하여 렉 없는 고속 검색 제공
   - **원문 자동 동기화**: 검색 결과 카드를 클릭하면 원문 뷰어 영역이 해당 라인 위치로 부드럽게 스크롤(Smooth Scroll)되며, 해당 라인을 1.5초간 노란색 테두리와 배경으로 시각화(Highlight)

4. **단독 실행 파일(.exe) 배포 자동화**
   - GitHub Actions와 PyInstaller를 연동하여 코드 푸시 시 클라우드 윈도우 환경에서 실행 파일(`app.exe`)이 자동 빌드되도록 세팅 완료

## 🛠️ 실행 및 빌드 가이드

### 1. 개발자 모드로 실행하기 (Local Run)
파이썬이 설치되어 있는 로컬 개발 환경에서 직접 실행하는 방법입니다.

```bash
# 1. 의존성 패키지 설치
uv add flask

# 2. Flask 서버 실행
cd program/code
uv run python app.py
```
* 서버 실행 후 브라우저에서 `http://127.0.0.1:5000`에 접속합니다.

---

### 2. 윈도우용 단독 실행 파일(.exe) 배포하기 (GitHub Actions)
파이썬이 없는 친구분 컴퓨터에 전달할 실행 파일을 제작하는 방법입니다.

1. 본 코드를 개인 GitHub 저장소에 푸시합니다.
2. 해당 GitHub 레포지토리의 **Actions** 탭으로 이동합니다.
3. **Build Standalone Windows Executable** 워크플로우 빌드가 성공적으로 끝나면, 해당 빌드 로그 하단의 **Artifacts**에서 **`windows-executable.zip`** 파일을 다운로드합니다.
4. 압축 파일 내부에 들어 있는 **`app.exe`** 파일만 친구에게 전달합니다.

> **친구 컴퓨터에서의 동작 방식:**
> - `app.exe`를 더블클릭하면 콘솔 창이 뜨지 않고 백그라운드에서 실행되며, 1초 후 자동으로 크롬 등 기본 인터넷 브라우저가 기동하여 검색 대시보드가 열립니다.
> - 업로드한 텍스트 파일 및 SQLite 데이터는 실행 파일과 같은 위치에 생성되는 `DB/` 폴더 내에 누적 보존됩니다.
