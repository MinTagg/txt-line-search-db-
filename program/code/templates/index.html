TXT Line Search DB ? 텍스트 파일(.txt, .hwp)을 줄 단위로 DB화하여, 복합 검색식(AND, OR, 괄호)으로 원하는 줄을 빠르게 찾을 수 있는 도구입니다.

텍스트 파일을 줄 단위로 저장하고 복잡한 검색식으로 한눈에 탐색하세요.
