# 목적

해당 프로그램은 txt 파일을 슬라이싱 -> DB화 -> and / or () 연산자등을 이용해서 검색을 하도록 하는 프로그램이다

폴더 구성
```
.
├── docs
│   └── READMD.md
└── program
    ├── code
    └── DB
        ├── DB
        │   ├── sample_1
        │   └── sample_2
        └── files
            ├── sample_1.txt
            └── sample_2.txt
```

sample_1.txt 예시 설명

1. texts = files/sample_1.txt 파일 불러오기
2. 줄 단위로 파일을 슬라이싱
5. DB/sample_1 폴더를 만들고 안에 DB 생성
6. DB는 단순하게 sqlite를 이용
7. 검색기능은 간단하게 되어야 함
검색 : (A and C) -> 문장 안에 A와 B가 동시에 있는 것 검색 => '<|SPACER|>ABCD\n', '<|SPACER|>IJKLABC' 반환

질문사항:
DB화 시킬 때, 해당 문장이 sample_1.txt 파일 기준으로 몇번째 줄에서 시작하는지도 추가할 수 있는지?
